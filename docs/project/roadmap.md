# Roadmap

Future development plans for the PHP K8s fork.

## Ongoing

- ✅ **PHP 8.3+ Support** - Continue leveraging latest PHP features
- ✅ **Kubernetes API Updates** - Support new Kubernetes versions as released
- ✅ **Documentation** - Comprehensive guides and examples
- ✅ **Bug Fixes** - Responsive issue resolution

## Planned Features

### Short Term

- 🔄 **Additional Resource Types** - Support for new Kubernetes resources
- 🔄 **Enhanced Examples** - More real-world usage patterns
- 🔄 **Performance Optimization** - Leverage PHP 8.3+ performance improvements

### Medium Term

- 📅 **PHP 8.4 Support** - Full compatibility with PHP 8.4
- 📅 **Extended API Reference** - Complete documentation for all classes
- 📅 **Improved Testing** - Enhanced test coverage

### Long Term

- 🎯 **Community Growth** - Build active community
- 🎯 **Plugin System** - Extensible architecture
- 🎯 **Advanced Features** - Based on community feedback
